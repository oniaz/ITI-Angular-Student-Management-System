# Student Management System

A simple Angular project built as an ITI Angular course task.

This app demonstrates core Angular concepts, including:
- Routing
- Components
- Services
- BehaviorSubject and Observable
- Parent-child communication with @Input and @Output

## Pages
- Home
- Students
- Add Student
- Student Details

## Run locally
1. Install dependencies: `npm install`
2. Start the app: `npm start`
3. Open: `http://localhost:4200`

## GitHub Pages deploy
Use:

`npm run deploy:gh`
